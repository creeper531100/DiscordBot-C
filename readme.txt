-*-addcbn-*-|addcbn [類別] [組合句]
```
組合句使用方法:
```
\addadv 新增一段組合句
\advlist 列出目前組合句
\addcbn 加入組合詞
\advcbn 列出目前組合句
```
addadv 使用方法:
```
>addadv 新增的句子，要;組合;的詞語
```
要隨機組合的句子加上分號(例如上句子的 ;組合; )，機器人會隨機抽出一筆資料來組合
例如:
```
\addadv ;髒話;，;人物;你不會來幫喔
```
-------------------------------------
機器人顯示: 靠杯，liu你不會來幫喔
```
如果想新增組合詞資料，請參考addcbn

addcbn 使用方法:
```
\addcbn 組合字典 組合詞
```
上句範例的"組合字典"是想新增的字典，而"組合詞"是想新的的詞語
例如:
```
\addadv 髒話 幹你娘
-------------------------------------
機器人可能會顯示: 
1. 靠杯，liu你不會來幫喔
2. 幹你娘，liu你不會來幫喔
```
